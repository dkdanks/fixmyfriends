# fixmyfreinds

Fresh Next.js app for `fixmyfreinds.com`.

## Getting started

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000).

## Current scope

- Marketing landing page for the group funding concept
- Messaging around practical birthday gifts with a reason
- Placeholder product framing for future fund creation, checkout, and payout flows
